For setup:

git init
git remote add origin https://github.com/Trolom/FootballAnalysisApp.git
git pull origin main

cd backend
python3 manage.py migrate
python3 manage.py runserver

cd frontend
npm install
npm run dev

cd backend
celery -A config.celery_app worker --loglevel=INFO --pool=solo

Kaggle dataset: https://www.kaggle.com/datasets/saberghaderi/-dfl-bundesliga-460-mp4-videos-in-30sec-csv

Get models from drive, both player and pitch models

Helpful for installing NVIDIA Container Toolkit:
https://www.server-world.info/en/note?os=Ubuntu_24.04&p=nvidia&f=3

After merging docker branch:

Laptop with GPU: docker compose up
Laptop without GPU: docker compose -f docker-compose.yml -f docker-compose.cpu.yml up
